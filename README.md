# Stackr API - Documentação

Documentação da API Stackr para hospedar, gerenciar e fazer backup dos seus apps.

## Desenvolvimento local

Instale o [Mintlify CLI](https://www.npmjs.com/package/mint):

```bash
npm i -g mint
```

Execute na raiz do projeto:

```bash
mint dev
```

Visualize em `http://localhost:3000`.

## Publicação

As alterações são implantadas automaticamente após o push para o branch padrão.
