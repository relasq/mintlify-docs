## 1. Obtenha seu API Token

Acesse o dashboard da Stackr e gere um token de API. Guarde-o em local seguro.

## 2. Teste a autenticação

Verifique se seu token funciona listando seus bots:

```bash
curl -X GET "https://api.stackr.lat/bots" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## 3. Faça upload de um bot

Envie um arquivo .zip contendo o código do seu bot:

```bash
curl -X POST "https://api.stackr.lat/bots/upload" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -F "file=@meu-bot.zip" \
  -F "botName=Meu Bot" \
  -F "language=nodejs" \
  -F "memory=256MB"
```

## 4. Controle o bot

Use o ID retornado para iniciar, parar ou reiniciar seu bot:

```bash
# Iniciar
curl -X POST "https://api.stackr.lat/bots/{id}/start" \
  -H "Authorization: Bearer SEU_API_TOKEN"

# Parar
curl -X POST "https://api.stackr.lat/bots/{id}/stop" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```
