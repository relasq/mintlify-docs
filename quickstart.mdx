## 1. Obtenha seu API Token

Acesse o dashboard da Stackr e gere um token de API. Guarde-o em local seguro.

## 2. Teste a autenticação

Verifique se seu token funciona listando seus apps:

```bash
curl -X GET "https://api.stackr.lat/apps" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## 3. Faça upload de um app

Envie um arquivo .zip contendo o código do seu app:

```bash
curl -X POST "https://api.stackr.lat/apps/upload" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -F "file=@meu-app.zip" \
  -F "appName=Meu App" \
  -F "language=nodejs" \
  -F "memory=256"
```

## 4. Controle o app

Use o ID retornado para iniciar, parar ou reiniciar seu app:

```bash
# Iniciar
curl -X POST "https://api.stackr.lat/apps/{id}/start" \
  -H "Authorization: Bearer SEU_API_TOKEN"

# Parar
curl -X POST "https://api.stackr.lat/apps/{id}/stop" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```
