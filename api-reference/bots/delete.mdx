---
title: "Deletar bot"
description: "Remove permanentemente um bot e seus dados"
openapi: POST /bots/{id}/delete
---

Remove permanentemente um bot e todos os seus dados. Esta ação não pode ser desfeita.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/bots/bot_abc123/delete" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "Bot deleted successfully"
}
```

<Warning>
A exclusão é permanente. Faça um backup antes se precisar preservar dados.
</Warning>
