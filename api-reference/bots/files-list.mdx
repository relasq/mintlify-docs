---
title: "Listar arquivos"
description: "Lista arquivos e pastas em um diretório do bot"
openapi: GET /bots/{id}/files
---

Lista arquivos e pastas em um diretório do sistema de arquivos do bot.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Query params

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `path` | string | Sim | Caminho do diretório (ex: /app) |

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/bots/bot_abc123/files?path=/app" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```
