---
title: "Detalhes do bot"
description: "Retorna informações detalhadas de um bot específico"
openapi: GET /bots/{id}
---

Retorna informações detalhadas de um bot específico.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/bots/bot_abc123" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "name": "Meu Bot",
  "status": "RUNNING",
  "memoryMb": 256,
  "language": "nodejs",
  "command": "npm start",
  "lastError": null
}
```
