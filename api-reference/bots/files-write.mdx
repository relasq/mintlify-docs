---
title: "Escrever arquivo"
description: "Cria ou sobrescreve o conteúdo de um arquivo"
openapi: PUT /bots/{id}/files/content
---

Cria ou sobrescreve o conteúdo de um arquivo no sistema de arquivos do bot.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Body (JSON)

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `path` | string | Sim | Caminho do arquivo |
| `content` | string | Sim | Conteúdo do arquivo |

## Exemplo cURL

```bash
curl -X PUT "https://api.stackr.lat/bots/bot_abc123/files/content" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "path": "/app/config.json",
    "content": "{\"key\": \"value\"}"
  }'
```
