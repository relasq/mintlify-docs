Para a execução de um bot em execução.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/bots/bot_abc123/stop" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "Bot stopped successfully"
}
```
