---
title: "Logs do bot"
description: "Retorna os últimos logs de execução do bot"
openapi: GET /bots/{id}/logs
---

Retorna os últimos logs de execução do bot para diagnóstico e debugging.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Query params

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `tail` | integer | 100 | Número de linhas a retornar |

## Exemplo cURL

```bash
# Últimas 100 linhas (padrão)
curl -X GET "https://api.stackr.lat/bots/bot_abc123/logs" \
  -H "Authorization: Bearer SEU_API_TOKEN"

# Últimas 500 linhas
curl -X GET "https://api.stackr.lat/bots/bot_abc123/logs?tail=500" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```
