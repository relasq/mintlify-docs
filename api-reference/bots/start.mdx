---
title: "Ligar bot"
description: "Inicia a execução de um bot"
openapi: POST /bots/{id}/start
---

Inicia a execução de um bot que está parado.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/bots/bot_abc123/start" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "Bot started successfully"
}
```
