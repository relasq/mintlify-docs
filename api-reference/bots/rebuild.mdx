title: "Reconstruir bot"

openapi: POST /bots/{id}/rebuild
---

Reconstrói o container do bot do zero usando a imagem base. Útil para resolver problemas de build ou ambiente corrompido.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/bots/bot_abc123/rebuild" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "Bot rebuilding..."
}
```
