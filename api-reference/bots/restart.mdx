---
title: "Reiniciar bot"
description: "Reinicia um bot em execução"
openapi: POST /bots/{id}/restart
---

Reinicia um bot em execução. Equivalente a parar e iniciar novamente.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do bot |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/bots/bot_abc123/restart" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "Bot restarting..."
}
```
