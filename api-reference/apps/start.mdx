---
title: "Ligar app"
description: "Inicia a execução de um bot"
openapi: POST /apps/{id}/start
---

Inicia a execução de um app que está parado.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/apps/app_abc123/start" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "Bot started successfully"
}
```
