---
title: "Listar apps"
description: "Retorna todos os apps do usuário"
openapi: GET /apps
---

Retorna todos os apps do usuário autenticado.

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/apps" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "App1",
    "status": "RUNNING",
    "memoryMb": 256,
    "ram": "50MB / 256MB"
  }
]
```
