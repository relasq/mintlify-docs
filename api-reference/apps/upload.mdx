---
title: "Upload de novo app"
description: "Envia um arquivo .zip contendo o código do app"
openapi: POST /apps/upload
---

Envia um arquivo .zip contendo o código do app para ser hospedado na Stackr.

## Body (multipart/form-data)

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `file` | Arquivo | Sim | Arquivo .zip (máx 50MB) |
| `botName` | string | Não | Nome do app |
| `language` | string | Não | Linguagem (ex: nodejs, python) |
| `command` | string | Não | Comando de inicialização |
| `memory` | string | Não | Limite de memória: 100, 256, 512, 1024, etc. |
| `slug` | string | Não | Slug único para APIs (ex: meu-app) |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/apps/upload" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -F "file=@meu-app.zip" \
  -F "botName=Meu App" \
  -F "language=nodejs" \
  -F "command=npm start" \
  -F "memory=256" \
  -F "slug=meu-app"
```

## Resposta

```json
{
  "botId": "123e4567-e89b-12d3-a456-426614174000",
  "message": "App uploaded successfully. Building...",
  "status": "RUNNING"
}
```
