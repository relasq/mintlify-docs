---
title: "Ler arquivo"
description: "Retorna o conteúdo de um arquivo específico"
openapi: GET /apps/{id}/files/content
---

Retorna o conteúdo de um arquivo específico do sistema de arquivos do app.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Query params

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `path` | string | Sim | Caminho do arquivo (ex: /app/index.js) |

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/apps/app_abc123/files/content?path=/app/index.js" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "content": "console.log('Hello');"
}
```
