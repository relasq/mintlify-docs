---
title: "Criar pasta"
description: "Cria uma nova pasta no sistema de arquivos do app"
openapi: POST /apps/{id}/files/mkdir
---

Cria uma nova pasta no sistema de arquivos do app.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Body (JSON)

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `path` | string | Sim | Caminho da nova pasta |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/apps/app_abc123/files/mkdir" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"path": "/app/data"}'
```

## Resposta

```json
{
  "success": true
}
```
