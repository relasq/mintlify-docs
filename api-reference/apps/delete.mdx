---
title: "Deletar app"
description: "Remove permanentemente um app e seus dados"
openapi: POST /apps/{id}/delete
---

Remove permanentemente um app e todos os seus dados. Esta ação não pode ser desfeita.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/apps/app_abc123/delete" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "App deleted successfully"
}
```

<Warning>
A exclusão é permanente. Faça um backup antes se precisar preservar dados.
</Warning>
