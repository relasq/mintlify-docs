---
title: "Reiniciar app"
description: "Reinicia um app em execução"
openapi: POST /apps/{id}/restart
---

Reinicia um app em execução. Equivalente a parar e iniciar novamente.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/apps/app_abc123/restart" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "message": "App restarting..."
}
```
