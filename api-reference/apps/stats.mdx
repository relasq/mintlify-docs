---
title: "Estatísticas em tempo real"
description: "Retorna CPU, RAM e uso de rede em tempo real"
openapi: GET /apps/{id}/stats
---

Retorna métricas em tempo real do app: CPU, RAM e rede.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/apps/app_abc123/stats" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
{
  "running": true,
  "cpu": "0.5%",
  "ram": "50MiB / 256MiB",
  "networkRx": "1KB"
}
```
