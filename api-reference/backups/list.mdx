---
title: "Listar backups"
description: "Lista todos os backups disponíveis"
openapi: GET /backups
---

Lista todos os backups disponíveis do usuário.

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/backups" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

```json
[
  {
    "id": "backup-uuid-1234",
    "appName": "Meu App",
    "createdAt": "2024-01-01T12:00:00Z",
    "sizeBytes": 5000000
  }
]
```
