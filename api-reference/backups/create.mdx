---
title: "Criar e baixar backup"
description: "Gera backup em tempo real e inicia download via streaming"
openapi: POST /apps/{id}/backup
---

Gera um backup em tempo real do app e inicia o download via streaming. O arquivo não é mantido no servidor.

## Parâmetros de path

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `id` | string | ID único do app |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/apps/app_abc123/backup" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -o backup.zip
```

## Resposta

Retorna o arquivo .zip diretamente via streaming. Use `-o` para salvar em disco.
