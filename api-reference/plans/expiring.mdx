---
title: "Planos expirando"
description: "Lista planos vencendo em breve"
openapi: GET /plans/expiring
---

Retorna a lista de planos que estão próximos do vencimento.

## Exemplo cURL

```bash
curl -X GET "https://api.stackr.lat/plans/expiring" \
  -H "Authorization: Bearer SEU_API_TOKEN"
```

## Resposta

Array de objetos com planos que expiram em breve.
