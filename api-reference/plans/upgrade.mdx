---
title: "Upgrade de plano"
description: "Adiciona mais memória ao plano atual"
openapi: POST /plans/upgrade
---

Adiciona memória adicional ao plano atual.

## Body (JSON)

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `additionalMemoryMb` | integer | Memória adicional em MB |
| `duration` | string | Período: `monthly` ou `yearly` |

## Exemplo cURL

```bash
curl -X POST "https://api.stackr.lat/plans/upgrade" \
  -H "Authorization: Bearer SEU_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "additionalMemoryMb": 256,
    "duration": "monthly"
  }'
```

## Resposta

Retorna os dados da transação PIX para pagamento do upgrade.
