## Sobre a Stackr API

A Stackr API permite que você hospede, gerencie, ligue/desligue e faça backups dos seus apps. Use a API para integrar apps ao seu fluxo de trabalho e automatizar operações.

## Base URL

Todas as requisições devem ser enviadas para:

```
https://api.stackr.lat
```

## Autenticação

Todas as rotas protegidas exigem um Token de API. Envie o token no header `Authorization`:

```
Authorization: Bearer <SEU_API_TOKEN>
```
